# Obra Fácil Pro

Sistema web mobile-first para apoio em obras, do levantamento ao acabamento.

## Pacote atual

**Pacote 16 — Vite + Supabase Ready**

Este pacote mantém o sistema funcionando no GitHub Pages e adiciona preparação para desenvolvimento moderno com Vite e futura integração Supabase.

## Rodar como site estático

Basta abrir `index.html` ou publicar no GitHub Pages.

## Rodar com Vite

```bash
npm install
npm run dev
```

## Verificar JavaScript

```bash
npm run check
```

## Preparação Supabase

Veja:

```txt
docs/SUPABASE_SETUP.md
backend/schema-inicial.sql
.env.example
```

## Aviso

O sistema auxilia levantamento, orçamento e planejamento, mas não substitui engenheiro, arquiteto ou profissional habilitado.
