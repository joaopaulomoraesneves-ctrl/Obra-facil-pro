# Obra Fácil Pro

Sistema web mobile-first para apoio em obras, do levantamento ao acabamento.

## Pacote atual

**Pacote 20 — Financeiro Real da Obra**

Este pacote adiciona:

- Recebimentos.
- Despesas reais.
- Parcelas.
- Contas a pagar.
- Fluxo de caixa.
- Lucro previsto vs lucro real.

## Rodar local

```bash
npm install
npm run dev
```

## Verificar

```bash
npm run production:check
```

## Aviso

O sistema auxilia levantamento, orçamento e planejamento, mas não substitui engenheiro, arquiteto, contador ou profissional habilitado.
