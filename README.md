# Obra Fácil Pro

Sistema web mobile-first para apoio em obras, do levantamento ao acabamento.

## Pacote atual

**Pacote 17 — Refinamento Pré-Backend**

Este pacote fortalece a base antes do Supabase real.

## Adicionado

- AppState.
- Migrações locais.
- Validação pré-backend.
- Classificação de alertas.
- Testes básicos de cálculo.
- Documentação da matriz matemática.

## Rodar local

```bash
npm install
npm run dev
```

## Verificar

```bash
npm run prebackend:check
```

## Aviso

O sistema auxilia levantamento, orçamento e planejamento, mas não substitui engenheiro, arquiteto ou profissional habilitado.
