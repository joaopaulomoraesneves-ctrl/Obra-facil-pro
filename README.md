# Obra Fácil Pro

Sistema web mobile-first para apoio em obras, do levantamento ao acabamento.

## Pacote atual

**Pacote 18 — Polimento Final Pré-Produção**

Este pacote melhora usabilidade, apresentação para cliente, onboarding, backup preventivo, testes e documentação antes do backend real.

## Rodar local

```bash
npm install
npm run dev
```

## Verificar

```bash
npm run production:check
```

## Publicar no GitHub Pages

Envie todos os arquivos para a raiz do repositório. O `index.html` continua funcionando como site estático.

## Aviso

O sistema auxilia levantamento, orçamento e planejamento, mas não substitui engenheiro, arquiteto ou profissional habilitado.
