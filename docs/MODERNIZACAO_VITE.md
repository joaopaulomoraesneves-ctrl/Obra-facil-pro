# Modernização com Vite

## Por que usar Vite?

Vite facilita:

- Servidor local de desenvolvimento.
- Build para produção.
- Futuro uso de módulos ES.
- Integração com Supabase.
- Possível migração futura para React.

## Importante

O sistema ainda funciona em GitHub Pages sem `npm install`, pois o `index.html` continua carregando arquivos JavaScript diretamente.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
npm run check
```
