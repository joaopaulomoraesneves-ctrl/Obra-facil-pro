# Matriz Matemática — Obra Fácil Pro

## Geometria

```txt
área = largura × comprimento
perímetro = 2 × (largura + comprimento)
parede bruta = perímetro × altura
parede útil = parede bruta - portas - janelas
volume = largura × comprimento × altura
```

## Materiais

```txt
caixas de piso = teto((área × (1 + perda)) ÷ rendimento por caixa)
litros de tinta = ((parede útil × demãos) ÷ rendimento) × (1 + perda)
blocos = teto(parede útil × blocos/m² × (1 + perda))
```

## Orçamento

```txt
base = materiais + mão de obra + despesas
segurança = base × margem de segurança
mínimo = base + segurança
lucro = mínimo × margem de lucro
final = mínimo + lucro - desconto
```

## Cronograma

```txt
dias por etapa = quantidade ÷ produtividade ajustada por equipe e complexidade
```

## Confiança

- Alta: área, perímetro, piso, parede, rodapé.
- Média: blocos, tinta, orçamento, cronograma e compras.
- Validação técnica obrigatória: elétrica, hidráulica e estrutura.
