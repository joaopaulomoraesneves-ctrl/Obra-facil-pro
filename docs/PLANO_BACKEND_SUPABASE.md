# Plano Backend/Supabase

## Objetivo
Migrar a versão local para uma versão com login, banco PostgreSQL e backup em nuvem.

## Etapas
1. Manter a versão local funcionando.
2. Exportar dados estruturados pela aba Backend.
3. Criar banco com o SQL inicial.
4. Adicionar autenticação.
5. Migrar obras.
6. Separar matriz matemática em módulos.
7. Adicionar assinatura/SaaS.

## Segurança futura
Ativar Row Level Security, políticas por usuário/tenant, backup automático e logs de erro.
