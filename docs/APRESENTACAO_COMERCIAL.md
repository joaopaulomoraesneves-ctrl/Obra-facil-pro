# Apresentação Comercial — Obra Fácil Pro

## O que é

Sistema web para auxiliar profissionais da construção no levantamento, orçamento, acompanhamento e organização de obras.

## Principais módulos

- Cadastro de obras e cômodos.
- Planta baixa 2D.
- Cálculo de materiais.
- Composições de serviço.
- Instalações básicas.
- Orçamento.
- Cronograma.
- Compras.
- Diário de obra.
- Relatório para PDF.
- Backup.
- Preparação para nuvem/SaaS.

## Valor para o cliente

Reduz esquecimentos, melhora a apresentação do orçamento, ajuda no controle de compras e organiza a execução da obra.
